# Superstore Sales Analysis Dashboard

## Overview
An interactive sales analysis dashboard built using Tableau Public, 
analysing 9,800 rows of retail sales data from 2015-2019.

## Live Dashboard
[View Dashboard Here](https://public.tableau.com/app/profile/akhilesh.udaya.kumar/viz/SuperstoreSalesAnalysisDashboard_17824276853970/SuperstoreSalesAnalysisDashboard)

## Key Insights
- West region leads sales at $710K — highest performing region
- Technology is the top category driving 36% of total revenue
- Phones and Chairs are top sub-categories at $330K combined
- Sales show consistent upward trend from 2015 to 2019

## Tools Used
- Tableau Public
- Microsoft Excel (data cleaning)

## Dataset
Superstore Sales Dataset — Kaggle (9,800 rows, 18 columns)
